⚡การคำนวณฟลักซ์ไฟฟ้า (Electric Flux)

พื้นฐานการคำนวณที่ต้องแม่น! นิยาม สูตร และทิศทาง

💡

0. ฟลักซ์ไฟฟ้าคืออะไร? (The Concept)

นิยามง่ายๆ: คือปริมาณที่บอกว่ามี "เส้นแรงไฟฟ้า" พุ่งทะลุผ่าน "พื้นที่หน้าต่าง" มากน้อยแค่ไหน

เปรียบเหมือน "น้ำฝน" (สนามไฟฟ้า $\vec{E}$) ที่สาดเข้า "หน้าต่าง" (พื้นที่ $\vec{A}$)
  • ฝนตกหนัก ($E$ มาก) $\rightarrow$ น้ำเข้าเยอะ
  • หน้าต่างบานใหญ่ ($A$ มาก) $\rightarrow$ น้ำเข้าเยอะ
  • เปิดหน้าต่างรับตรงๆ ($\theta = 0^\circ$) $\rightarrow$ น้ำเข้าเต็มที่!
📐

1. สูตรหลักที่ต้องใช้ (The Formula)


สมการฟลักซ์ไฟฟ้า:

$\Phi_E = EA \cos \theta$

• $\Phi_E$ = ฟลักซ์ไฟฟ้า (N·m²/C)
• $E$ = ขนาดสนามไฟฟ้า (N/C)
• $A$ = พื้นที่ผิว (m²)
• $\theta$ = มุมระหว่างสนามไฟฟ้า $\vec{E}$ กับ เส้นแนวฉาก (Normal Vector)

⚠️ ระวัง! ถ้าโจทย์ให้มุมกับ "ระนาบแผ่น" ต้องเอา $90^\circ$ ลบออกเสมอ!

👁️

2. เช็กทิศทางและมุม $\theta$ (จุดตาย)


  • ตั้งฉากผิว ($\theta = 0^\circ$): ฟลักซ์สูงสุด ($\Phi = EA$) เส้นแรงทะลุเต็มๆ
  • ขนานผิว ($\theta = 90^\circ$): ฟลักซ์เป็นศูนย์ ($\Phi = 0$) เส้นแรงแค่เฉียดผ่าน
  • ทำมุมเฉียง: ต้องแตกแรง $\vec{E}$ เข้าหาเส้นปกติเสมอ (ใช้ $\cos \theta$)
➕➖

3. เครื่องหมายบอกทิศทาง (Sign Convention)

เมื่อพิจารณา "ผิวปิด" (เช่น ทรงกลม หรือ กล่อง):
  • ฟลักซ์เป็นลบ (-): เมื่อเส้นแรงไฟฟ้าพุ่ง "เข้า" สู่ผิว (Inward)
  • ฟลักซ์เป็นบวก (+): เมื่อเส้นแรงไฟฟ้าพุ่ง "ออก" จากผิว (Outward)
🎯

4. เมื่อโจทย์มาเป็นเวกเตอร์ (Vector Form)

หากโจทย์ให้ $\vec{E} = E_x\hat{i} + E_y\hat{j} + E_z\hat{k}$ และ $\vec{A} = A_x\hat{i} + A_y\hat{j} + A_z\hat{k}$

ใช้ Dot Product:
$\Phi_E = \vec{E} \cdot \vec{A} = (E_x A_x) + (E_y A_y) + (E_z A_z)$

*คูณเฉพาะตัวที่แกนเดียวกันแล้วจับบวกกัน
